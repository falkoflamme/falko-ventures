## DESIGN-VISION — SO MUSS DIESE SEITE SICH ANFÜHLEN

### Das Gefühl:
Stell dir vor, Apple baut eine Keynote-Website für einen europäischen Venture Builder. Stell dir vor, Y Combinator Demo Day hat eine Aftershow-Page. Stell dir vor, ein Investor öffnet diese Seite auf seinem iPad im Flieger und kann nicht aufhören zu scrollen.

**DAS** ist das Gefühl.

### Design-DNA:
- **Cinematic** — Jede Section ist eine Filmszene, nicht eine Webseite
- **Confident** — Die Zahlen sprechen. Das Design lässt sie sprechen.
- **Breathing** — Großzügiger Raum, aber KEIN toter Raum. Jeder Pixel hat Zweck.
- **Alive** — Subtile Animationen die zeigen: hier passiert etwas. Aber nie circusy.
- **Dark** — Fast schwarz. Gold als Signalfarbe. Das ist kein Blog, das ist eine Investment-Opportunity.

### Typografie:
- **Display:** Großer Serif für Headlines — Cormorant Garamond oder Playfair Display. Elegant, zeitlos, premium.
- **Body:** Sora oder DM Sans — clean, modern, lesbar
- **Mono:** JetBrains Mono — für Zahlen, Labels, Tags, Status-Badges
- **VERBOTEN:** Inter, Roboto, Arial, System-Fonts

### Farben:
```
--void: #050208          (tiefster Hintergrund)
--deep: #0B0716          (Section-Hintergrund)
--surface: #110E1D       (Karten, erhöhte Flächen)
--gold: #C9A84C          (Signature-Farbe, CTAs, Highlights)
--gold-light: #E8D5A0    (Gold-Text auf dunklem Grund)
--accent: #6C4FF0        (Sekundär-Akzent, Purple)
--white: #F5F3EE         (Haupttext)
--gray: #8B8898          (Sekundärtext)
--muted: #4A4660         (Tertiärtext, Labels)
```

### Animationen:
- **Scroll-triggered Fade-Up** — mit IntersectionObserver. KRITISCH: Fallback opacity:1 wenn JS nicht lädt. Content darf NIE unsichtbar bleiben.
- **Counter-Animation** — Zahlen zählen hoch wenn sie in den Viewport kommen
- **Hotel-Logo Marquee** — infinite scroll: Rocco Forte · JW Marriott · Peninsula · Sofitel · Kloster Eberbach · Ruby Hotels
- **Hover-States** — Karten heben sich, Gold-Border erscheint
- **Progress Bar** — Gold-Gradient am oberen Rand, zeigt Scroll-Position
- **Subtle Grain** — leichte Noise-Textur über dem gesamten Background
- **Parallax auf Hero-Bild** — dezent, nicht motion-sick

### Layout:
- Full-Viewport Sections (jede Section = ein Argument)
- Max-Width 1200px für Content
- 2-Spalten Grid für Ventures (Desktop), 1 Spalte Mobile
- Sticky transparent Nav mit Scroll-Effekt (wird solid bei Scroll)
- Mobile: Hamburger Menu, Touch-optimiert

## TECHNISCHE ANWEISUNGEN

### Animations — KRITISCH:
```css
/* JEDES animierte Element braucht dieses Pattern: */
.fade-in {
  opacity: 1; /* FALLBACK — Content ist IMMER sichtbar */
}
@media (prefers-reduced-motion: no-preference) {
  .fade-in {
    opacity: 0;
    transform: translateY(20px);
    transition: opacity 0.6s ease, transform 0.6s ease;
  }
  .fade-in.visible {
    opacity: 1;
    transform: translateY(0);
  }
}
```
IntersectionObserver triggert `.visible`. Wenn JS nicht lädt → Content trotzdem da.
